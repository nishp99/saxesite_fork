Code for On Random Weights and Unsupervised Feature Learning

This package contains matlab code for reproducing results reported in

Saxe, A., Koh, P.W., Chen, Z., Bhand, M., Suresh, B., & Ng, A. (2011). On random weights and unsupervised feature learning. In ICML 2011. (pdf)

Saxe, A., Koh, P.W., Chen, Z., Bhand, M., Suresh, B., & Ng, A. (2010). On random weights and unsupervised feature learning. In NIPS 2010 Workshop on Deep Learning and Unsupervised Feature Learning. (pdf) (supplementary material).

Many thanks to Quoc Le and Jiquan Ngiam for permission to bundle their excellent TCNN code.
